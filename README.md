# TaskFlow Backend 🚀

A professional Spring Boot backend project with JWT Authentication and Task Management APIs.

## Features
- User Registration
- User Login
- JWT Authentication
- Create Task
- Update Task
- Delete Task
- Get All Tasks
- Validation Handling
- Exception Handling
- Spring Security
- MySQL Database Integration

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- JWT
- Hibernate / JPA
- MySQL
- Maven

## API Endpoints

### Auth APIs
- POST /api/auth/register
- POST /api/auth/login

### Task APIs
- GET /api/tasks
- POST /api/tasks
- PUT /api/tasks/{id}
- DELETE /api/tasks/{id}

## Project Structure
- Controller Layer
- Service Layer
- Repository Layer
- DTO Layer
- Security Layer
